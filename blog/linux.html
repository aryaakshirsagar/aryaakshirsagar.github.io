Linux Notes — Commands I Actually Use

These are Linux notes — the ones I type almost without thinking.

Navigation — Don’t Get Lost in the Void

  • pwd → “Where am I?” *Existential crisis avoided.*
  • ls → Show me what’s here.
  • ls -la → Show me everything even the files pretending they don’t exist.
  • cd <dir> → Teleport into a folder.
  • cd .. → Retreat one step like a cautious penguin.
  • cd ~ → Home sweet home.

Making & Breaking Things

  • mkdir new_dir → Create a shiny new folder.
  • touch file.txt → Spawn an empty file that will eventually be replaced.
  • rm file.txt → Delete. Regret. Repeat.
  • rm -r folder → Nuke a folder from orbit.
  • mv old.txt new.txt → Rename it and pretend it’s a different file.
  • cp file.txt copy.txt → Duplicate it, because paranoia is healthy.
