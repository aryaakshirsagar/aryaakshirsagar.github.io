Linux: Commands I Actually Use

A practical quick-reference guide to the Linux commands I use daily — for system navigation, file manipulation, and a bit of network work.

Everyday Commands

  • ls -la → Show me everything, even the files pretending they don’t exist.
  • cd - → Jump back to where I just was. Instant teleport.
  • grep -r "text" . → Search inside files like a digital detective.
  • chmod +x script.sh → Give a script the green light to run.
  • sudo !! → Rerun the last command with superpowers.

File & System Management

  • df -h → Check how much space is left before panic sets in.
  • du -sh * → Find out which folders are hogging all the space.
  • tar -czvf archive.tar.gz folder/ → Compress stuff neatly.
  • htop → See what's eating your CPU (and end it).

Networking

  • ifconfig or ip a → Show network interfaces.
  • ping -c 4 example.com → See if the internet is alive.
  • netstat -tulnp → List open ports and services.

CTF Quick Tips

  • Always start with strings on binaries. Hidden messages love to hide there.
  • Use binwalk for files — you might find embedded treasures.
  • Don’t forget exiftool for images; metadata can be surprisingly chatty.
  • Bookmark CyberChef. It’s like a Swiss Army knife for encoding, decoding, and weird transformations.
