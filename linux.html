Linux Notes

These are notes the ones I type almost without thinking.

Navigation

  • pwd : “Where am I?”
  • ls : Show me what’s here.
    • ls -la : Show me everything even the files pretending they don’t exist.
  • cd <dir> : Teleport into a folder.
  • cd .. : Retreat one step like a cautious penguin.
  • cd ~ : Home sweet home.

Making & Breaking Things

  • mkdir new_dir : Create a shiny new folder.
  • touch file.txt : Spawn an empty file that will eventually be replaced.
  • rm file.txt : Delete. Regret. Repeat.
  • rm -r folder : Nuke a folder from orbit.
  • mv old.txt new.txt : Rename it and pretend it’s a different file.
  • cp file.txt copy.txt : Duplicate it, because paranoia is healthy.

Finding Stuff Without Clicking Around

  • find / -name "*.txt" : Search for all .txt files starting from root.
  • grep "keyword" file.txt : Search inside a file like an X-ray.
  • grep -r "password" /etc : Reckless.
  • locate something : The lazy cousin of find.

Reading Files Like a Hacker

  • cat file.txt : Dump everything at once.
  • less file.txt : Scroll .
  • head file.txt : Just the first 10 lines.
  • tail file.txt : The last 10 lines.
  • tail -f logfile.log : Live stream your system’s drama in real-time.

Permissions & Ownership

  • chmod 755 script.sh : Give it permission to run.
  • chmod +x file.sh : Make it executable.
  • chown user:group file.txt : Claim it as your own.
  • sudo : Borrow root’s superpowers (temporarily, hopefully).

Package Management

Debian/Ubuntu gang:

  • sudo apt update : Update the pantry list.
  • sudo apt upgrade : Upgrade everything (pray nothing breaks).
  • sudo apt install neofetch : For flexing your OS in screenshots.

Fedora/CentOS gang:

  • sudo dnf install htop : See processes in HD.

Process Mind Control

  • ps aux : See everything that’s alive.
  • top : Watch your CPU freak out in real time.
  • kill <pid> : Pull the plug on a process.
  • kill -9 <pid> : Pull the plug hard.
  • htop : The cooler, colorized cousin of top.

Networking Commands

  • ip a : Show all network interfaces.
  • ping google.com : Are we online?
  • curl ifconfig.me : What’s my public IP?
  • netstat -tulpn : See which apps are listening.
  • ss -tulpn : The newer, faster version of netstat.

Quick

  • !! : Run the last command again (great for sudo oopsies).
  • history : See every command you’ve typed .
  • clear : Hide your shame from the terminal.
  • df -h : Show disk space .
  • du -sh * : See folder sizes in current directory.
  • watch -n 1 <command> : Run a command every second .

sudo commads can be a demon.
